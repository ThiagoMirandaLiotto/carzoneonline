    Asistencia

    Dirección

    El Huracán EVO RWD cuenta con Electronic Power Steering (EPS), es decir, con una dirección asistida electrónica que se adapta a todas las modalidades de conducción del sistema ANIMA para garantizar la mejor asistencia de viraje, tanto en carretera como en pista.

    Mecanismo

    Transmisión

    Imagine poder utilizar un mecanismo de doble fricción en la misma unidad: cuando se selecciona una marcha, se engrana la siguiente para reducir al mínimo el tiempo de embrague/desembrague a través de un mecanismo que permite obtener la máxima reactividad y continuidad.

    Directrices

    Sistema Ruedas

    El Huracán EVO ofrece una combinación única entre el sistema activo de dirección (Lamborghini Dynamic Steering) y el de dirección trasera (Lamborghini All-wheel Steering) para ofrecer un rendimiento dinámico de máximo nivel.
